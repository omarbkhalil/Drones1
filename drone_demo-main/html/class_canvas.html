Detailed Description

The Canvas class provides a widget that is capable of drawing and managing geometric shapes, drones, and Voronoi diagrams.

This class integrates functionalities for drawing polygons, triangles, Voronoi diagrams, and drone images. It supports interactive features such as mouse events to manipulate the shapes and elements on the canvas.

Canvas::Canvas ( QWidget * parent = nullptr)
explicit

Constructs a new Canvas object.

Initializes the Canvas with a parent QWidget. The Canvas is configured to track mouse events and optionally display various graphical elements like triangles, the Delaunay triangulation, and the centers of triangles. It also loads a drone image that is used within the graphical representation.
